# SafeMeal

급식 사진과 식사 기록을 바탕으로 개인별 섭취 현황과 권장 배식량을 제공하는 스마트 급식 관리 웹 서비스입니다.

학생은 오늘의 식단과 알레르기 정보를 확인하고, 식사 전·후 사진을 등록해 메뉴별 섭취 결과를 확인할 수 있습니다. 관리자는 메뉴와 일자별 식단을 관리하고 운영 통계와 잔반 현황을 조회할 수 있습니다.

## 주요 기능

### 학생

- 로그인 및 사용자 정보 확인
- 오늘의 급식 메뉴와 알레르기 정보 조회
- 개인별 권장 배식량 확인
- 식사 전·후 사진 등록 및 분석 요청
- 메뉴별 섭취량과 영양 정보 확인
- 분석 결과의 섭취 비율 보정
- 최근 식사 기록 조회

### 관리자

- 날짜별 급식 운영 현황 조회
- 메뉴별 평균 섭취율과 잔반율 확인
- 메뉴 등록 및 수정
- 일자별 점심 식단 구성

## 서비스 흐름

1. RFID 인식으로 사용자의 식사 기록을 생성합니다.
2. 사용자가 식사 전·후 사진을 등록합니다.
3. 백엔드가 사진을 분석해 메뉴별 섭취 비율과 영양 정보를 저장합니다.
4. 사용자는 분석 결과와 다음 배식 권장량을 확인합니다.
5. 관리자는 누적 기록을 바탕으로 급식 및 잔반 통계를 확인합니다.

## 기술 구성

- HTML, Vanilla JavaScript
- Tailwind CSS
- Vite
- FastAPI 기반 REST API 연동
- Vercel 배포

프론트엔드는 화면 표시와 사용자 입력을 담당합니다. 인증, 데이터 저장, 이미지 분석, 섭취량 계산 및 권장량 산출은 백엔드 API에서 처리합니다.

## 화면 구성

- 로그인
- 사용자 대시보드
- 식사 사진 등록
- 식사 기록
- 식사 분석 상세
- 관리자 대시보드
